# 11st_Assignment

## 실행 스크립트

<hr>

```
npm install package.json

parcel index.html
```

<br>

## 폴더 구조

<hr>

```bash
├── README.md              - 리드미 파일
├── index.html             - 최상단 파일
├── src                    - 어플리케이션 폴더
│   ├── Assets
│   │    └── Image         - 로컬 이미지 폴더
│   │
│   ├── components         - 컴포넌트 및 scss 폴더
│   │    ├─ AlarmComponent
│   │    ├─ ClockComponent
│   │    ├─ ImageComponent
│   │    ├─ MainComponent
│   │    └─ MemoComponent
│   │
│   │── core               - 추상화 컴포넌트 폴더
│   └── js                 - 라우터 및 기능 정의 js 폴더 
│  
└── App.js
```

## 특이사항
